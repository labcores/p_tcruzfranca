Brazilian Protest 2013, the Gold Standard Dataset

This page presents a sentiment-annotated Twitter gold standard dataset for the Brazilian protests in 2013. The dataset consists of 4,422 messages (tweets) annotated by three raters with information about the sentiment expressed in the messsages. This is a valuable resource for social media-based sentiment analysis in the context of protests events. Besides the gold standard dataset, we are also provide the raw dataset as eight files of tweets IDs and a python script for retrieve the tweets.

Download and Preparation

Preparation

You will need to prepare a some things (a few configurations):

  1. Install a python 2.7 interpreter and the pip package management.
  2. Clone the repositorygit clone https://github.com/tcruzfranca/annotatedDatasetBrazilianProtests.git or download it.
  3. Now, into your local repository
  4. Install some dependencies (libs): Install required dependecies: pip install -r dependencies.txt --upgrade
  5. Create or use a Twitter App credentials (a read-only Access Token) and then fill appropriately the "__main__" in the end of the recoveryTweetsById.py
Some details about the gold standard dataset

The tweets in the dataset received either "P", "N" or "NN" labels for Positive, Negative or Neutral messages, respectively. The gold standard is a csv file with: tweets' IDs (first column), the annotation of the raters (columns 2, 3 and 4) and the consolidation result is in column 5. The first line has the informations about the license and the second the fields names. Using the python script provided by us, you can retrive the dataset messages. However, if you want see see the csv dataset file, visit the source on git hub. We have followed the Twitter’s Terms of Service.

The python script to retrieval tweets

That script is a python 2.7 script that can be seen here. You should just to follow the instructions above: create Twitter credentials, copy and paste the credentials in the appropriate fields and run.

Som raw dataset details

The raw dataset is a set of csv files (click here to see the files). Those files have just the tweets' IDs. In order to retrieval the tweets, you should adapt the python script in the "__main__" to read such files and save the tweets. We have followed the Twitter’s Terms of Service.
